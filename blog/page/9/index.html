Hibernate中的继承关系

Hibernate中支持3种形式实现继承关系：
1. Table per concrete class 表与子类之间独立一对一关系
2. Table per subclass 每个子类对应一张子表，并与主类共享主表
3. Table per class hierarchy 表与类一对多关系

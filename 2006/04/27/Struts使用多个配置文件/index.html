StrutsдҪҝз”ЁеӨҡдёӘй…ҚзҪ®ж–Үд»¶

Posted on 2006-04-27 In еҗҺз«Ҝ

жіЁпјҡжң¬ж–ҮжҳҜеҜ№гҖҠдҪҝз”ЁеӨҡдёӘStruts й…ҚзҪ®ж–Үд»¶гҖӢдёҖж–Үзҡ„з¬”и®°пјҢж–№дҫҝи®°еҝҶе’ҢжҹҘиҜўгҖӮ

1. дҪҝз”ЁеӨҡдёӘstruts-config.xml ж–Үд»¶:

еҪ“struts-config.xml еҸҳеҫ—и¶ҠжқҘи¶ҠеӨ§ж—¶пјҢз»ҙжҠӨиҝҷдёӘж–Үд»¶еҸҳжҲҗдәҶдёҖд»¶еҫҲз—ӣиӢҰзҡ„дәӢгҖӮеҸҜд»Ҙж №жҚ®дёҚеҗҢзҡ„еҠҹиғҪжҲ–жЁЎеқ—е°Ҷstruts-config.xml еҲҶи§ЈдёәдёҚеҗҢзҡ„ж–Үд»¶пјҢд»Ҙж–№дҫҝеҗҺжңҹзҡ„ејҖеҸ‘е’Ңз»ҙжҠӨгҖӮ

з¬¬дёҖжӯҘ

жҢүз…§еҠҹиғҪе°Ҷstruts-config.xml еҲ’еҲҶдёәеҮ дёӘе°Ҹж–Үд»¶пјҢе…·дҪ“еҒҡжі•жҳҜпјҡ
(1) е°Ҷеә”з”ЁзЁӢеәҸе…¬е…ұзҡ„е…ЁеұҖй…ҚзҪ®ж”ҫеҲ°struts-config.xml дёӯпјӣ
(2) е°ҶзӣёеҗҢеҠҹиғҪзҡ„й…ҚзҪ®ж”ҫеҲ°еҗҢдёҖдёӘй…ҚзҪ®ж–Үд»¶дёӯпјҢеҰӮпјҡstruts-sms.xml,struts-mms.xml,struts-report.xmlгҖӮе»әи®®жүҖжңүзҡ„й…ҚзҪ®ж–Үд»¶йғҪд»Ҙstruts-еҒҡеүҚзјҖпјҢе№¶дё”еңЁж–Үд»¶еҗҚдёӯиЎЁжҳҺеҠҹиғҪжЁЎеқ—пјӣ

з¬¬дәҢжӯҘ

й…ҚзҪ®web.xmlеј•з”ЁеӨҡдёӘй…ҚзҪ®ж–Үд»¶пјҡ

жіЁж„Ҹпјҡж–Үд»¶й—ҙз”ЁйҖ—еҸ·еҲҶйҡ”пјҢеҢ…еҗ«жҜҸдёӘж–Үд»¶зҡ„е®Ңж•ҙи·Ҝеҫ„гҖӮ

2. дҪҝз”ЁеӨҡдёӘиө„жәҗж–Үд»¶пјҡ

еҗҢж ·зҡ„еҺҹеӣ пјҢе®һзҺ°еӨҡиҜӯиЁҖзҡ„иө„жәҗж–Үд»¶еӨӘеӨ§д№ҹжҳҜдёҖд»¶з—ӣиӢҰзҡ„дәӢжғ…пјҢжҲ‘д»¬д№ҹеҸҜд»Ҙе°Ҷе®ғжҢүз…§еҠҹиғҪеҲҶи§ЈдёәеҮ дёӘж–Үд»¶иҝӣиЎҢз»ҙжҠӨгҖӮ

з¬¬дёҖжӯҘ

жҢүз…§еҠҹиғҪе°ҶеҶ…е®№ж”ҫе…ҘдёҚеҗҢзҡ„иө„жәҗж–Үд»¶пјҢдёҚеӨҡиөҳиҝ°пјӣ

з¬¬дәҢжӯҘ

еңЁstruts-config.xmlдёӯж”ҫзҪ®иө„жәҗж–Үд»¶пјҡ
< message-resources parameter =вҖқResourcesвҖқ null =вҖқfalseвҖқ />
< message-resources parameter =вҖқSmsResourcesвҖқ key =вҖқsmsвҖқ />
< message-resources parameter =вҖқMmsResourcesвҖқ key =вҖқmmsвҖқ />
< message-resources parameter =вҖқReportResourcesвҖқ key =вҖқreportвҖқ />

иҜҙжҳҺпјҡиҝҷйҮҢйқўзҡ„з¬¬дёҖдёӘе…ғзҙ пјҲжІЎжңү key еұһжҖ§пјүжҲҗдёәй»ҳи®Өзҡ„иө„жәҗйӣҶгҖӮжүҖд»ҘжүҖжңүзҡ„ JSP е’Ң servet йғҪеҸҜд»ҘйҖҡиҝҮ servlet дёҠдёӢж–Үи®ҝй—®иҝҷз»„иө„жәҗгҖӮе…¶д»–зҡ„иө„жәҗеҲҷйңҖиҰҒйҖҡиҝҮжҢҮе®ҡеҗҺйқўзҡ„keyеҖјжқҘи®ҝй—®иө„жәҗгҖӮ

з¬¬дёүжӯҘ

еңЁйЎөйқўдёӯеј•з”Ёиө„жәҗж–Үд»¶пјҡ
< bean:message bundle =вҖқsmsвҖқ key =вҖқsms.label.send.nameвҖқ />

иҜҙжҳҺпјҡеұһжҖ§ bundleз”ЁдәҺжҢҮе®ҡиҰҒдҪҝз”Ёзҡ„йқһй»ҳи®ӨеҢ…пјҢеҚіжҢҮе®ҡеүҚйқўеңЁstruts-config.xmlдёӯзҡ„keyеҖјгҖӮ
